# **App Name**: Batch Image Manager

## Core Features:

- Login Authentication: Secure login page to authenticate users via username and password.
- Image Listing: Lazy-loaded image gallery to efficiently display a large number of images. Displays images from the API.
- Image Deletion: Delete images via the API one at a time.
- Individual Image Download: Download individual images directly.
- Batch Image Selection: Enable users to select multiple images for batch actions.
- Batch Image Download: Download selected images one after the other without zipping them.
- Long Press Selection: Enable long-press selection of images on touch screen devices.

## Style Guidelines:

- Primary color: Deep sky blue (#3EB489) to communicate focus and efficiency.
- Background color: Very light blue (#E0F7FA), nearly white, creating a calm background.
- Accent color: A deep slightly greenish cyan color (#008080) provides contrast for key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and readable interface.
- Use a grid layout for the image gallery to maximize content display.
- Simple, minimalist icons for actions like delete and download.
- Subtle fade-in animations as images load, providing smooth transitions.
